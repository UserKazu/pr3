# Практична робота №3 — Розгортання Application Server на базі NodeJS та Fastify Виконав: студент групи КН-41 — Пловайко Вадим Це готовий до запуску приклад проєкту для практичної роботи №3. Проєкт містить Fastify-сервер з простим файловим сховищем для ресурсів і приклад конфігурації Docker + Nginx. ## Запуск локально (без Docker) 1. Встановіть Node.js (рекомендовано v18+ або актуальну LTS). 2. `npm install` 3. `npm run dev` (для розробки) або `npm start` 4. API доступне за адресою: `http://localhost:3000/api/resources` ## Запуск з Docker Compose 1. `docker compose up --build` 2. Відкрийте `http://localhost/api/resources` (Nginx на порту 80 проксіює на застосунок). ## Структура проекту - `src/server.js` — старт Fastify - `src/routes/resources.js` — CRUD-ендпоінти - `data/resources.json` — просте файлове сховищe - `nginx/default.conf` — конфігурація Nginx (проксі) - `Dockerfile`, `docker-compose.yml` — для контейнеризації ## Примітки - Сховище реалізовано файлами для простоти навчальної роботи — для реального проєкту краще використовувати базу даних.
